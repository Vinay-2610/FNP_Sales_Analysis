# 🌸 FNP Sales Analysis Dashboard

## 📌 Project Overview

This project presents a **Sales Analysis Dashboard** for Ferns N Petals (FNP) using Excel tools like **Power Query, Power Pivot, and DAX**.
The dashboard provides insights into sales performance, customer behavior, and order trends.

---

## 🎯 Problem Statement

The objective of this project is to analyze sales data and generate meaningful insights such as:

* Total revenue and order trends
* Customer spending patterns
* Top-performing products and cities
* Seasonal and occasion-based sales trends

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Power Query (Data Cleaning & Transformation)
* Power Pivot (Data Modeling)
* DAX (Data Analysis Expressions)
* Pivot Tables & Charts

---

## 📂 Project Files

* 📊 `FNP_Sales_Data.csv` → Raw dataset
* 📸 `Dashboard.png` → Final dashboard visualization
* 📄 `Problem_Statement.pdf` → Project problem description

---

## 📈 Dashboard Features

* ✅ Total Orders, Revenue, and Avg Customer Spending (KPIs)
* 📅 Revenue analysis by Months and Days
* 🎉 Revenue by Occasions (Diwali, Birthday, Anniversary, etc.)
* 🏙️ Top 10 Cities by Orders
* 🛍️ Top 5 Products by Revenue
* 🔍 Interactive filters (Date & Occasion slicers)

---

## 🔑 Key Insights

* Peak sales observed during **festive occasions**
* Certain cities contribute significantly to total orders
* A few products generate the majority of revenue
* Customer spending varies across different seasons

---

## 🚀 How to Use

1. Download the dataset (`.csv`)
2. Open Excel and load data using Power Query
3. Build relationships using Power Pivot
4. Create measures using DAX
5. View or customize the dashboard
